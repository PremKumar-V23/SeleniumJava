# 🚀 Selenium Java Automation Framework

A Selenium WebDriver automation project built using **Java**, **Maven**, and **TestNG**. This repository documents my journey of learning and implementing industry-standard UI automation testing practices.

---

## 📖 About

This project is part of my QA Automation learning path. It focuses on building a scalable Selenium automation framework while exploring TestNG, Page Object Model (POM), reporting, and CI/CD integration.

## ▶️ Run the Project

Clone the repository:

```bash
git clone https://github.com/PremKumar-V23/SeleniumJava.git
```

Navigate to the project:

```bash
cd SeleniumJava
```

Install dependencies:

```bash
mvn clean install
```

Run all tests:

```bash
mvn test
```
